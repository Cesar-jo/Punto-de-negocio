Preguntas frecuentes

En este apartado encontrara las preguntas mas frecuentes como tambien podra resolver alguna de sus dudad.

¿Qué es punto de negocio?

Punto de negocio es un sistema ERP enfocado en la administración, gestión, control de ventas, reportes, analisis de datos para una mejor administración de su negocio, implementando IA, administración total para sus oficinas u otros establecimientos donde el cliente podra rentar o vender los dicho mencionados, en caso de un usuario ordinario que requiera una renta o venta, podra ver en el sistema para obtener informacion de los locales y poder hacer la compra o renta de ellos.

¿Cómo obtengo el sistema?

Antes de comprar algun plan es nesesario averse registrado para que ya tenga una cuenta hecha, una vez adquirido el plan adecuado y aver procesado el pago, se le redireccionara otra vez al login para que inicie session, pero ahora como usuario admninistrador donde podra subir sus locales, oficinas u otros establecimientos para que pueda administrar todas las ventas y tener una buena gestión y control de ello.

¿Qué pasa si no se paga a tiempo el plan tomado?

Automaticamente su cuenta quedara como un usuario normal, hasta que se vuelva a pagar se reactivara su cuenta como admnistrador.
